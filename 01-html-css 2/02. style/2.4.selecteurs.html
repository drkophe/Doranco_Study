Les sélecteurs

Il existe plusieurs moyens de cibler (sélectionner) des éléments html via le css. On peut cibler les éléments par :

  • leur nom de balise
  • leur nom de classe
  • leur id
  • une hiérarchie (chemin)
  • ...
