Les balises HTML

Les titres (headings)

Les balises h1 à h6 représentent les niveaux de titre d'un document HTML, h1 étant unique sur une même page et le plus impotant. Les différents niveaux suivants (h2 à h6) servent à hiérarchiser et structurer l'information au sein de la page.

Paragraphe et mise en forme

La balise "p" permet de créer un paragraphe.

Certaines balises permettent de mettre en évidence une partie de texte et d'en modifier la mise en forme. "strong" = marquer l'importance d'un texte. "b" = bold = afficher du texte en gras. Mais "b" est obsolète. "s" = strike = marquer le caractère obsolète de quelque chose.

"em" = emphase = autre manière de mettre en évidence du texte. "i" = exprimer un terme technique, du texte en langue étrangère ou encore un pensée.

"br" = Permet un saut de ligne. J'ai sauté une ligne dans mon éditeur mais je ne vois pas de saut de ligne sur la page web.
Grace au "br", j'ai sauté une ligne sur ma page. Attention, "br" ne doit pas être utilisé par espacer des élements entre eux. Généralement, il sert de retour à la ligne au sein d'un même élément. "br" est une balise auto-fermante, c'est à dire quelle n'a pas besoin d'être refermée car elle n'a pas vocation à intégrer du contenu. Son rôle se limite à un saut de ligne.

"u" = underline = permet de souligner du texte.

Les types (catégories) d'éléments

On recense 3 grandes catégories d'éléments en HTML :

  1. block
  2. inline
  3. inline-block

Spécificités des éléments de type block

  • Par défaut, un élément de type block occupe toute la largeur de son conteneur (parent).
  • Par défaut, un élément de type block s'adapte en hauteur à son contenu.
  • par défaut,mles éléments de type block s'alignent les uns en-dessous des autres.
  • Les dimensions d'un élément de type block peuvent être redéfinies.

Spécificités des éléments de type inline

  • Largeur : largeur du contenu

  • Alignement: les uns à côté des autres
  • Les dimensions d'un élément de type inline ne peuvent pas être redéfinies.

  • Redimensionnement : possible
